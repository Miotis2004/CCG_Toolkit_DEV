# Card Authoring

This Phase 0 skeleton defines conventions for future implementation.

## Scope

- Runtime code lives under `Assets/CCGToolkit/Runtime` in the `CCGToolkit` namespace.
- Editor-only tools live under `Assets/CCGToolkit/Editor` in the `CCGToolkit.Editor` namespace.
- Tests live under `Assets/CCGToolkit/Tests/EditMode` and `Assets/CCGToolkit/Tests/PlayMode`.
